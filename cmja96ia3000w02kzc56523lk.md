# Deux failles critiques dans Chrome : mettez votre navigateur à jour !

## TL;DR

- Deux vulnérabilités critiques de Chrome (CVE-2025-14765 et CVE-2025-14766) permettent à des attaques à distance via des pages web malveillantes.
- Ces failles touchent la mémoire via WebGPU (Use-After-Free) et le moteur V8 de JavaScript.
- Mettez à jour Chrome vers les versions **143.0.7499.146/.147** ou supérieures pour éviter ces risques.
- Accédez au menu paramètres de Chrome pour télécharger et appliquer la mise à jour.

## Pourquoi ces failles sont-elles critiques ?

Google Chrome est l’un des navigateurs les plus utilisés, avec plus de 3,4 milliards de personnes dans le monde, ce qui en fait une cible privilégiée pour les cyberattaques. Les failles CVE-2025-14765 et CVE-2025-14766 représentent un risque majeur, car elles permettent potentiellement à un attaquant d'exécuter du code à distance simplement en incitant l’utilisateur à visiter une page web malveillante.

Ces types de vulnérabilités, qui n'exigent aucune interaction de l'utilisateur après l'accès à la page compromise, sont particulièrement dangereuses. Elles peuvent être exploitées dans des campagnes de piratage massif visant autant des individus que des entreprises.

## Comment mettre à jour Chrome ?

Pour protéger votre système, il est impératif de mettre à jour votre navigateur Chrome dès que possible. Voici une procédure simple pour vous assurer que vous utilisez une version corrigée :

1. Cliquez sur le menu **Plus** (les trois points verticaux en haut à droite).
2. Accédez à **Paramètres**, puis à **À propos de Chrome**.
3. Si une mise à jour est disponible, Chrome effectuera le téléchargement automatiquement.
4. Redémarrez le navigateur pour activer les changements.

### Versions sécurisées :
- **Windows et macOS** : 143.0.7499.146 / 143.0.7499.147
- **Linux** : 143.0.7499.146

Assurez-vous que votre version est bien celle indiquée afin de bénéficier de la protection.

[Découvrez un guide complet pour mettre à jour Chrome](https://www.malwarebytes.com/blog/explained/2025/06/how-to-update-chrome-on-every-operating-system).

## Détails techniques des failles découvertes

### CVE-2025-14765 : Vulnérabilité WebGPU (Use-After-Free)

Le système WebGPU, conçu pour prendre en charge les graphiques avancés, les jeux et les calculs intenses tels que certaines applications d’IA, est à l’origine de cette faille. Voici les principales caractéristiques :

- **Origine** : Cette API graphique accède à des ressources spécifiques à la mémoire du système.
- **Exploitation** : Une page HTML malveillante peut provoquer une corruption de la mémoire en accédant à une zone libérée, ce qui ouvre la porte à l’exécution de code arbitraire.
- **Raison technique** : La vulnérabilité est due à une gestion incorrecte de la mémoire, un scénario classique pour les failles de type Use-After-Free.

### CVE-2025-14766 : Vulnérabilité dans le moteur JavaScript V8

Le moteur V8, utilisé pour exécuter le JavaScript au cœur de Chrome, est également vulnérable :

- **Origine** : La faille provient d’erreurs dans les accès à la mémoire dans V8, permettant de sortir des zones autorisées.
- **Exploitation** : Les attaquants peuvent injecter du code malveillant via un contenu HTML ou JavaScript spécialement conçu.
- **Impact** : Cette faille peut permettre des actions de lecture et d’écriture hors des limites de la mémoire, conduisant à un contrôle total sur le système visé.

Ces deux vulnérabilités combinées renforcent l’importance de corriger les failles rapidement avant qu’elles ne soient exploitées dans des campagnes larges.

## Importance des mises à jour régulières

Les failles critiques comme celles-ci soulignent l'importance des mises à jour régulières :

- **Protection proactive** : Les mises à jour corrigent les vulnérabilités avant qu'elles ne puissent être exploitées à grande échelle.
- **Réduction des risques** : Les campagnes de piratage ciblent souvent les versions non corrigées des logiciels. Mettre à jour diminue significativement les chances d’être affecté.
- **Automatisation** : Activez les mises à jour automatiques dans Chrome afin de garantir que votre navigateur reste aligné sur les dernières versions sans action manuelle.

Les navigateurs basés sur Chromium (Edge, Brave, etc.) peuvent également présenter des vulnérabilités similaires. Surveillez leurs mises à jour pour garantir une protection efficace.

## Protégez-vous contre les menaces

Face aux failles critiques comme celles-ci, voici quelques conseils pratiques pour rester en sécurité :

- **Effectuez les mises à jour immédiatement** : Le correctif est disponible et peut être appliqué en quelques minutes.
- **Activez les mises à jour automatiques** : La meilleure sécurité est de rendre le processus automatique pour réduire les délais de réaction.
- **Vérifiez régulièrement vos logiciels** : Les navigateurs, plugins ou outils hérités peuvent contenir des vulnérabilités similaires. Réalisez des audits réguliers.
- **Surveillez les campagnes d’exploitation active** : Les cyberattaques suivent de près les annonces de failles critiques. Il est essentiel de rester informé pour agir rapidement si nécessaire.

Chrome reste une cible populaire en raison de sa large adoption, mais ces correctifs montrent que Google prend les menaces au sérieux. N’attendez pas pour mettre à jour votre navigateur. Agir rapidement est le meilleur moyen de garantir votre sécurité en ligne.
