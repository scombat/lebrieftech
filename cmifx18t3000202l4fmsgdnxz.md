# Applications Linux ressuscitées par la communauté open source

## TL;DR

- La communauté open source revitalise des logiciels Linux abandonnés, leur offrant une deuxième vie sous des formes modernisées.
- 11 applications telles que `iotop-c`, `vokoscreenNG` et `Shutter` ont été mises à jour avec des adaptations aux technologies actuelles comme Wayland et Qt6.
- Ces projets soulignent l'importance de la collaboration communautaire pour préserver et améliorer les outils essentiels.
- Contribuer à ces projets, par le biais de signalements de bugs, de développements ou de dons, permet d'assurer leur pérennité.

## L'importance de l'open source dans la résurrection logicielle

L'open source joue un rôle fondamental en permettant la préservation des logiciels qui, bien que souvent laissés à l'abandon, restent utiles. Lorsque des mainteneurs cessent de travailler sur un projet ou que de nouvelles technologies rendent une application obsolète, ce n'est pas nécessairement la fin. Grâce à l'engagement des développeurs passionnés, ces logiciels connaissent une seconde vie. Les forks et réécritures sont des solutions couramment utilisées pour revivifier ces outils, en les rendant conformes aux standards modernes et aux technologies émergentes.

La communauté open source agit souvent comme un catalyseur pour prolonger la pertinence des applications. Qu'il s'agisse de répondre à de nouveaux besoins technologiques, de corriger l'absence de mainteneurs ou d'améliorer des fonctionnalités initiales, cette dynamique contribue à créer un écosystème logiciel plus robuste, adaptable et durable.

## Applications Linux remises au goût du jour : Exemples

### **iotop-c**
- **Description** : Cet outil, réécrit en C, est une version améliorée de `iotop`. Il permet de surveiller les entrées/sorties (I/O) des disques avec plus de rapidité et d'efficacité que son prédécesseur en Python.
- **Installation** : `sudo apt install iotop-c`

### **vokoscreenNG**
- **Description** : Successeur de `vokoscreen`, cet enregistreur d'écran utilise GStreamer et Qt, tout en étant compatible avec Wayland.
- **Installation** : `sudo apt install vokoscreen-ng` ou via Flathub.

### **WoeUSB-ng**
- **Description** : Une refonte en Python de `WoeUSB`, conçue pour faciliter la création de clés USB bootables Windows. Malgré ses améliorations, son développement semble actuellement en pause.
- **Installation** : Utilisateurs Arch Linux : `yay -S woeusb-ng`

### **eSpeak NG**
- **Description** : Fork du moteur de synthèse vocale `eSpeak`, il intègre davantage de langues et de fonctionnalités.
- **Installation** : `sudo apt install espeak-ng`

### **stress-ng**
- **Description** : Une version modernisée de `stress`, qui permet de simuler des charges importantes pour tester la durabilité matérielle et logicielle. **Attention** : Utilisation à des fins de test uniquement.
- **Installation** : `sudo apt install stress-ng`

### **aircrack-ng**
- **Description** : Suite mise à jour pour effectuer des tests d'intrusion sur réseaux WiFi sécurisés par WPA ou WEP. Ce projet étend les fonctionnalités d'`aircrack` original.
- **Installation** : `sudo apt install aircrack-ng`

### **tomboy-ng**
- **Description** : Réécriture légère et efficace en Pascal de l'outil de prise de notes `Tomboy`. Il remplace l'alternative initiale en C# tout en conservant la compatibilité avec les anciens fichiers.
- **Installation** : `sudo apt install tomboy-ng`

### **radiotray-ng**
- **Description** : Version modernisée en C++ de l'application minimaliste `Radiotray`. Ce lecteur de radio en ligne simplifie l'accès audio via la barre système.
- **Installation** : Utilisateurs Fedora : `sudo dnf install radiotray-ng`

### **GoldenDict-ng**
- **Description** : Adaptation de GoldenDict avec des nouvelles fonctionnalités telles que la prise en charge étendue de formats, un mode sombre, et une intégration avec Qt6.
- **Installation** : `sudo apt install goldendict-ng` ou via Flathub.

### **ntopng**
- **Description** : Remplacement de l'analyseur réseau `ntop` avec une interface utilisateur moderne et des fonctions avancées comme l'inspection des paquets.
- **Installation** : `sudo apt install ntop-ng`

### **Shutter**
- **Description** : Application classique de capture d'écran adaptée pour fonctionner avec le serveur d'affichage Wayland.
- **Installation** : `sudo apt install shutter`

## À retenir : Pourquoi soutenir ces projets

La capacité de la communauté open source à ressusciter des applications Linux abandonnées est l'exemple parfait de l'innovation collective. Ces projets garantissent non seulement la pérennité de solutions indispensables mais permettent également de préserver une culture de partage et d'itération constante dans le développement logiciel. 

Pour les utilisateurs, soutenir ces initiatives revêt plusieurs formes : signalement de bugs, développement actif ou contribution financière. Ce soutien assure la continuité des projets et leur adaptation aux besoins de demain.

En conclusion, l'open source ne se contente pas d'adapter l'ancien au présent, il construit les bases d'un futur plus stable, collaboratif et inclusif pour l'industrie technologique.
