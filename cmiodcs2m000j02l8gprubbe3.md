# ONLYOFFICE Docs 9.2 : Découvrez les innovations clés !

## TL;DR

- Vérification orthographique et grammaticale assistée par l'IA intégrée.
- Automatisation des tâches grâce à la création de macros.
- Raccourcis clavier personnalisables pour un flux de travail optimisé.
- Améliorations pour l'édition des formulaires avec des étiquettes descriptives et des fonctionnalités collaboratives.
- Nouveaux outils pour les PDF, comme les couleurs de suppression pour mieux organiser les modifications.

## Nouveautés de ONLYOFFICE Docs 9.2

La mise à jour 9.2 d’ONLYOFFICE Docs représente un pas significatif dans l’évolution des suites bureautiques open source. Conçu pour les équipes collaboratives et les utilisateurs individuels, cet outil répond aux exigences croissantes en matière de productivité, personnalisation et gestion documentaire.

### Automatisation avec les macros

Les macros constituent une avancée majeure pour automatiser les tâches répétitives dans l’environnement bureautique. Cette fonctionnalité permet aux utilisateurs de programmer des actions spécifiques telles que :

- Le formatage automatique des textes.
- La manipulation de données dans des tableaux.
- Toute autre tâche récurrente nécessitant une exécution rapide et fiable.

Pour créer une macro :
1. Accédez à l'onglet **Affichage**.
2. Sélectionnez l'option **Enregistrer une macro**.
3. Enregistrez vos actions, puis réexécutez-les à tout moment pour automatiser votre flux de travail.

Ces macros peuvent être particulièrement utiles pour les utilisateurs techniques souhaitant introduire une logique répétitive dans leurs processus.

### Personnalisation des raccourcis clavier

ONLYOFFICE Docs 9.2 introduit une nouvelle fonctionnalité attendue : la personnalisation des raccourcis clavier. Elle offre la possibilité d’adapter entièrement la navigation de l’application à vos préférences personnelles ou à vos habitudes acquises dans d’autres logiciels.

Comment configurer vos raccourcis clavier :
1. Rendez-vous dans **Fichier → Paramètres avancés**.
2. Configurez les raccourcis en fonction de votre usage.
3. Testez et affinez votre configuration pour améliorer vos workflows.

Cela permet une navigation plus fluide et une gestion simplifiée, particulièrement intéressante pour les professionnels travaillant intensivement sur de gros projets documentaires.

### Vérification grammaticale boostée par IA

L’un des points forts de cette mise à jour est l’intégration d’un outil d’intelligence artificielle pour l’analyse grammaticale et orthographique. Ce plugin IA permet de scanner vos documents pour identifier et corriger les erreurs linguistiques.

Fonctionnalités principales :
- Accès via l'onglet **AI** ou le menu contextuel (clic droit).
- Analyse de sections spécifiques ou du document entier.
- Suggestions détaillées avec explications pour chaque correction.

Cette fonctionnalité s’avère cruciale pour les utilisateurs rédigeant des documents formels ou techniques, où la précision est primordiale. Bien qu’efficace, elle pourrait avoir des limites sur les cas linguistiques complexes.

### Améliorations pour les formulaires et PDF

ONLYOFFICE Docs 9.2 présente plusieurs nouveautés pour l’édition de formulaires et la gestion des fichiers PDF :

**Ajouts pour les formulaires** :
- Étiquettes descriptives pour les cases à cocher, boutons radios et autres champs interactifs.
- Attribution de rôles spécifiques aux champs de formulaire pour une meilleure collaboration dans les projets d’équipe.

**Outils pour les PDF** :
- Application de couleurs aux suppressions dans les PDF, facilitant la différenciation visuelle des modifications.
- Outil utile pour catégoriser les fichiers ou gérer des droits spécifiques.

Ces fonctionnalités améliorent significativement l’édition et la collaboration sur des documents complexes.

## À Surveiller : Risques et Enjeux

Bien que cette mise à jour soit riche en innovations, certains aspects méritent une attention particulière :

- **Modèles IA** : La précision et la pertinence du correcteur grammatical dépendent largement de la qualité des modèles IA intégrés.
- **Programmation des macros** : L’efficacité des macros repose sur une configuration correcte. Une erreur pourrait entraîner des échecs dans l’automatisation.
- **Adoption des nouvelles fonctionnalités** : Les utilisateurs habitués à l’ancienne configuration devront peut-être investir du temps pour s’adapter.

Que vous soyez développeur, ingénieur ou gestionnaire documentaire, cette mise à jour offre des outils modernes et efficaces pour répondre à vos besoins.
