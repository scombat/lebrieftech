# Cinnamon 6.6 : Une mise à jour clé pour Linux Mint

## TL;DR

- Refonte du menu avec de nouvelles catégories et un positionnement optimisé des boutons.
- Apparition des badges de notification dans la barre des tâches pour un suivi simplifié.
- Nouvelle organisation des thèmes pour une personnalisation accrue.
- Ajout d'une applet dédiée à la lumière nocturne.
- Compatibilité avec Linux Mint 22.3 Zena et disponibilité via distributions rolling release.

## Quoi de neuf dans Cinnamon 6.6 ?

### Changements majeurs dans le menu

Le menu de Cinnamon a été entièrement revisité pour offrir une expérience utilisateur plus fluide et intuitive. Parmi les nouveautés principales :

- Des **catégories spécifiques** placées au bas du menu pour simplifier la navigation.
- Les **boutons système** ont été déplacés en haut, ce qui permet de les retrouver plus rapidement.
- Vous pouvez désormais afficher des avatars optionnels et personnaliser les couleurs des boutons système.
- Les icônes peuvent être configurées pour utiliser un mode symbolique ou des couleurs selon les préférences.
- Un **nouveau pack d’icônes** appelé « xapp-symbolic-icons » est également disponible, renforçant la cohérence visuelle.

### Améliorations de l'interface et des fonctionnalités

La version 6.6 introduit plusieurs améliorations notables qui rendent l'environnement de bureau encore plus pratique pour les utilisateurs :

- **Animations fluides** : Le remplacement de Tweener par easing permet des transitions visuelles plus agréables.
- **Applet lumière nocturne** : Une fonctionnalité dédiée avec un mode "toujours activé" pour réduire la fatigue oculaire.
- **Système de notifications** : Les badges de notification sont désormais intégrés dans la barre des tâches et gèrent mieux les listes de fenêtres.
- Les miniatures des commutateurs d'espaces de travail affichent désormais les icônes des fenêtres pour une identification rapide.
- **Applet de contrôle sonore** : Les corrections apportées permettent d'éviter la disparition des pochettes d’album après les ajustements de volume.

Ces changements renforcent l'expérience utilisateur globale, en combinant esthétisme et ergonomie.

### Autres modifications significatives

Outre les modifications visuelles et fonctionnelles, Cinnamon 6.6 apporte quelques ajustements qui simplifient l’utilisation quotidienne :

- **Raccourci clavier** : Un nouveau raccourci permet de réinitialiser le zoom en un clic, idéal pour les utilisateurs multitâches.
- **Authentifications administratives** : Polkit a été amélioré pour simplifier les interactions nécessitant des droits système.
- **Calendriers distants** : Ces derniers bénéficient désormais d’un rafraîchissement automatique dès leur ouverture.
- **Gestion des fenêtres** : Les préférences associées ont été déplacées vers un nouvel onglet, simplifiant leur accès et leur personnalisation.

## Télécharger Cinnamon 6.6

Cinnamon 6.6 sera inclus dans la prochaine version de Linux Mint 22.3 « Zena ». Toutefois, pour les utilisateurs impatients ou ceux qui utilisent des distributions rolling release, plusieurs options sont disponibles dès maintenant :

- **Compiler à partir de la source** : Accédez au [dépôt GitHub de Cinnamon](https://github.com/linuxmint/cinnamon/releases/tag/master.mint22) pour obtenir le code source et l’installer manuellement.
- **Rolling release** : Des distributions comme EndeavourOS, CachyOS ou Arch Linux proposent déjà cette dernière version.

Ces alternatives garantissent une exploration rapide des nouvelles fonctionnalités sans attendre la mise à jour officielle de Linux Mint.

## À surveiller avec cette nouvelle version

Malgré ses améliorations significatives, Cinnamon 6.6 nécessite quelques précautions :

- Les changements dans le menu, notamment le déplacement des boutons système et l’introduction de nouvelles catégories, pourraient dérouter certains utilisateurs habitués à l’ancienne disposition.
- Les nouvelles fonctionnalités, notamment l’applet de lumière nocturne et les badges de notification, doivent être testées sur différents matériels pour éviter tout problème d’intégration ou de compatibilité.

Ces points d'attention rappellent l'importance d’un suivi actif par la communauté et les développeurs pour maintenir un environnement stable et performant.

Cinnamon 6.6 marque une étape majeure dans l'évolution de l'environnement de bureau de Linux Mint. Avec sa refonte du menu, ses fonctionnalités pratiques comme les badges de notification et l'appui sur l'organisation des thèmes, il s'impose comme une mise à jour essentielle pour les utilisateurs. Compatible avec Linux Mint 22.3 Zena, cette version est également accessible dès maintenant via plusieurs distributions rolling release.

Explorez dès aujourd'hui Cinnamon 6.6 pour tirer parti d’un environnement de bureau moderne et optimisé !
